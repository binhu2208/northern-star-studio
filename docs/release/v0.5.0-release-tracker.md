## Go / No-Go Notes

### Go if
- No blocker issues in the current build
- Build feels stable in a short fresh-session smoke test
- Release notes and known issues are accurate

### Hold if
- Save/progression/combat flow breaks
- Critical encounter or card logic is unreliable
- A likely day-one hotfix is already known but unverified
