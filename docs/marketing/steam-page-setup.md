# Steam Page Setup Checklist — Emotion Cards

This document outlines everything needed to launch the Steam store page for Emotion Cards, from asset requirements to manual steps Bin must complete on Steam.

---

## 1. Steam Store Page Requirements

### Capsules (Required)
| Asset | Dimensions | Format | Notes |
|-------|------------|--------|-------|
| Small Capsule | 460 × 215 | PNG | Shown in search results |
| Main Capsule | 460 × 215 | PNG | Primary store visibility |
| Large Capsule | 460 × 215 | PNG | Library and featured spots |

### Screenshots (Required — min 5, max 12)
- Minimum **5 screenshots** required
- Recommended: **8–10** for full showcase
- Dimensions: **1280 × 720** or **1920 × 1080** (16:9)
- Include a mix of:
  - Core gameplay moments
  - UI/card close-ups
  - Progression/reward moments
  - Menu/options screen
  - Replayability features (daily challenges, etc.)

### Trailer (Required)
- **Upload at least 1 trailer** (gameplay or announcement)
- Recommended length: 30s – 2min
- Format: MP4 or MOV
- Resolution: 1280×720 minimum, 1920×1080 preferred
- Include subtitles (captions burned in recommended)

### Header Image
- Dimensions: **460 × 215** (or 460 × 215 @2x for retina)
- Displayed on store page above the fold

## 3. Pricing Page Setup

### Base Pricing
1. Navigate to **Steamworks > Apps > Emotion Cards > Pricing**
2. Select **Set New Price**
3. Choose base currency (recommended: USD)
4. Set regional pricing defaults

### Recommended Price Tier
- **Emotion Cards** is a digital card game
- Suggested range: **$4.99 – $9.99** USD
- Consider:
  - Competitor pricing (Slay the Spire: $24.99, Inscryption: $14.99)
  - Game scope and replayability
  - Launch discount strategy

---

## 4. Region Pricing Configuration

Steam allows per-region pricing. Recommended approach:

### Tier 1 (Launch Regions)
| Region | Price (USD equiv.) |
|--------|-------------------|
| United States | $7.99 |
| Canada | $8.49 |
| United Kingdom | £5.99 |
| European Union | €7.99 |
| Japan | ¥980 |
| South Korea | ₩9,500 |
| Australia | $11.99 |

### Tier 2 (Discount Regions)
- Set to ~20–30% below Tier 1 for emerging markets
- Use Steam's **"Recommend Prices"** tool for guidance

### Steps
1. Go to **Pricing > Price List**
2. Click **Add New Price Schedule**
3. Assign regions to each tier
4. Test with Steam's **"Preview Store"** tool

## 6. Manual Actions Required (Bin)

These steps must be done manually by Bin on the Steam platform:

### Steamworks Dashboard
1. **Create App** — If not already created in Steamworks
2. **Upload Store Assets** — Capsules, screenshots, trailer (via Depot or manual upload)
3. **Fill Store Page Sections**:
   - Short Description (one-liner)
   - Full Description
   - Highlights
   - System Requirements
   - Release Notes
4. **Configure Pricing** — Set base price and regional tiers
5. **Set Visibility** — Switch from "Hidden" to "Coming Soon" or "Released"
6. **Configure Steam Curated** — Opt-in/opt-out of Steam features

### Community & Support
1. **Set up Steam Community** — Configure forums or link to Discord
2. **Add Support Contact** — Email or Discord link
3. **Upload Legal** — Privacy policy URL

### Pre-Launch (1–2 Weeks Before)
1. **Submit for Review** — Steam reviews store pages before going live
2. **Announce on Steam** — Post in Steam Announcements (optional)
3. **Monitor Wishlist** — Track growth, adjust marketing
