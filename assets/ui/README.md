# UI Assets

This folder contains production UI art for Emotion Cards.

## Structure

- `card-art/` — card illustration pipeline files, placeholders, review material, and finals
- `elements/` — reusable UI elements such as buttons and related specs/scripts
- `hud/` — persistent gameplay HUD assets and specs
- `TASKS.md` — active UI art task tracking

## Rules

- Keep placeholder art clearly labeled.
- Keep specs close to the assets they describe.
- Avoid mixing review boards with final implementation exports without folder labels.
- Preserve stable filenames already used by the game unless implementation is updated in sync.
