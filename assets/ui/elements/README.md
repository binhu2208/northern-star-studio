# UI Elements

Reusable interface art lives here.

## Typical contents

- button exports
- icon or state variants
- generator scripts
- element specs

## Working rules

- Keep generated helpers and production exports understandable at a glance.
- Store one nearby spec or README when naming, dimensions, or states matter.
- Use review files only when needed; final exports should stay easy for dev to identify.
