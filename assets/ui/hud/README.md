# HUD Assets

This folder contains persistent gameplay UI assets such as bars, indicators, and status icon groups.

## Expected contents

- production-ready HUD exports
- HUD sizing/spec references
- any review notes needed for implementation checks

## Working rules

- Keep filenames stable once integrated.
- Put dimensional or crop requirements in a nearby spec.
- If a HUD asset is replaced visually without a filename change, flag it for QA.
