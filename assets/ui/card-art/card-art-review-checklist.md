## Creative Checks

- [ ] Emotional read is clear in under 1 second
- [ ] Composition has one obvious focal point
- [ ] Silhouette is distinct from nearby cards in the same batch
- [ ] Character identity reads correctly
- [ ] Emotion family palette is correct
- [ ] Lighting direction supports the mood
- [ ] Background supports the focal point instead of competing with it
- [ ] Illustration remains readable at card size

## Technical Checks

- [ ] Source file uses approved naming convention
- [ ] Export file uses approved naming convention
- [ ] 600×400 implementation export exists
- [ ] 1200×800 hi-res export exists
- [ ] Card crop preserves the focal point
- [ ] No temp layers, guides, or editor UI are visible
- [ ] File opens correctly and is not corrupted
