# Assets Index

This folder holds production-facing art assets for Emotion Cards.

## Structure

- `characters/` — character portrait production, batch scaffolding, reviews, and exports
- `ui/` — card art, HUD assets, buttons, and other interface art

## Working rules

- Keep approved exports separate from review material.
- Use lowercase kebab-case filenames.
- Do not drop vague files like `final.png` or `latest.png` into shared folders.
- Put process docs and review references near the assets they support.

For full organization, naming, review, and handoff rules, see:
- `docs/art/asset-organization.md`
