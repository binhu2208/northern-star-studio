# Character Assets

Character folders are organized for portrait production in repeatable batches.

## Per-character structure

Each character folder should keep this layout:

- `batch-01-base/` — neutral base portrait and lineup work
- `batch-02-expressions/` — emotional variants and acting passes
- `batch-03-finish/` — polish, FX, and final rendering work
- `review/` — contact sheets, notes, markups, approval comps
- `exports/` — approved implementation-ready PNGs
- `README.md` or `SPEC.md` — identity, palette lane, and current status

## Rules

- Keep WIP and review files out of `exports/`.
- Preserve character-specific specs where they already exist.
- If a character is not fully defined yet, scaffold the folder but do not invent canon.
