# Flower Boat — Sprite Assets

All sprite assets for the Flower Boat Godot project.

## Directory Structure

| Directory | Contents | Status |
|-----------|----------|--------|
| `characters/` | Customer silhouette sprites (8 types) | Pending |
| `flowers/` | Flower illustrations (7 types) | Pending |
| `boat/` | Boat body, counter, sign | Pending |
| `environment/` | Canal water tiles, sky panels, dock markers | Pending |
| `weather/` | Weather overlay layers (rain, fog, golden) | Pending |
| `ui/` | Card frames, buttons, inventory slots | Pending |

## Quality Bar

**Non-negotiable:**
- Flowers look genuinely good — identifiable, fresh, not generic
- Boat has character — visible wood grain, worn but maintained
- Customer silhouettes read emotion through posture alone — no faces

## Naming Convention

`{category}_{variant}.png`

Examples:
- `flower_sunflower.png`
- `customer_hurry.png`
- `canal_water_sunny.png`
- `overlay_rain.png`
